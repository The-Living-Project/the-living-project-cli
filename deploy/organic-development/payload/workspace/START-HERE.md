# Start Here

This workspace is ready to use with any capable AI assistant.

## Universal Prompt

Paste this into your AI tool:

```text
Use The Living Project framework in this folder. Treat this as my active workspace.
Read `.living-project/QUICKSTART.md`, figure out the right phase automatically,
ask me only the minimum questions needed, and write the outputs into
`.living-project` as we go.
```

If you already know the problem you want to solve, paste it right under that prompt.

## Codex Shortcut

If the local Codex skill was installed, you can use:

```text
Use $living-project in this folder and guide me to the right next step.
```

## What Happens Next

1. The assistant picks the right phase.
2. It creates or updates the framework files for the project.
3. It keeps moving the project forward inside `.living-project/`.
