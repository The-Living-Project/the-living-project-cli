# The Living Project Quick Start

## Zero-Training Start

1. Open this folder in your AI tool.
2. Paste the prompt from `START-HERE.md`.
3. Answer the questions you get back.

That is enough to begin.

## What This Workspace Contains

- `seeds/` for project seed statements
- `context/` for context briefs and working memory
- `compost/` for post-mortems and reusable learnings
- `phases/` for the framework guidance used by the assistant
- `cultivate-log.md` for cross-project review and prioritization

## How The Framework Behaves

- If the project is brand new, it starts in `SEED`.
- If context already exists, it moves into the next likely phase.
- If the project needs review, reset, or prioritization, it routes to the right strategic phase.

## Universal Prompt

```text
Use The Living Project framework in this folder and guide me to the right next step.
```

## Codex Shortcut

If the local Codex skill is installed:

```text
Use $living-project in this folder and guide me to the right next step.
```
